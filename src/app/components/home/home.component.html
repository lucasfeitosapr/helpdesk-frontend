Help Desk Teste

Olá! Seja bem vindo ao sistema Help Desk desenvolvido por mim no curso Desenvolvedor Full Stack.
Esse é um sistema para gerenciamento de chamados simulando um simples Help Desk desenvolvido em Angular 12 e Spring Boot 2.
O sistema atualmente roda na nuvem, sendo o front-end na Vercel e o back-end na Heroku.
O projeto tem como objetivo principal abordar os conceitos de aplicações SPAs, Autenticação e Autorização com Tokens JWT entre outras funcionalidades básicas do Angular. O código fonte do projeto pode ser acessado clicando no link Github no menu lateral.

Fique a vontade para testar o sistema e deixar sua sugestão!

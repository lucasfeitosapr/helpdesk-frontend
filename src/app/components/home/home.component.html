Help Desk Suporte e Dicas

Olá! Seja bem-vindo ao sistema Help Desk, um produto desenvolvido durante o curso de Desenvolvimento Full Stack. Este sistema, que simula uma plataforma simplificada de Help Desk, foi construído utilizando as tecnologias Angular 12 e Spring Boot 2. Atualmente, o sistema está hospedado na nuvem, com o front-end na Vercel e o back-end na Heroku.

O objetivo principal deste projeto é demonstrar a implementação de conceitos fundamentais de Single Page Applications (SPAs), autenticação e autorização com Tokens JWT, além de outras funcionalidades básicas do Angular. O código fonte do projeto está disponível para consulta e pode ser acessado por meio do link do Github presente no menu lateral.

Sinta-se à vontade para testar o sistema e compartilhar suas sugestões!
